Module Units

Module Assignments

Assignment: System Design for Humanoid Robot Operations

This assignment involved the design of a Python-based object-oriented system to simulate a humanoid robot. It included UML diagrams (class, activity, sequence, state), and the use of stacks, queues, and lists for simulation logic.

Reflection: Helped me develop system-level thinking and understand how to translate requirements into design components.

Assignment: Implementation of the Humanoid Robot System

This implementation translated the system design into working Python code. It included a modular robot class, a CLI in main.py, and a test suite in test_robot.py.

Reflection: Strengthened my understanding of modularity, CLI design, and test-driven development. It was a rewarding end-to-end development challenge.

Final Project Evaluation

The final project was a Python simulation system for a humanoid robot. Its modular structure enabled realistic command execution and feedback through a CLI. I used unit testing and separation of concerns to ensure quality and maintainability.

Challenges included time constraints and simplifying some originally planned features. However, I gained deep insight into planning, coding, and testing in real-world-style projects.
